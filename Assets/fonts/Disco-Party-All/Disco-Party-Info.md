# Disco Party
Disco Party is a 24pt pixel art font designed with curvy and chunky 80s style lettering.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 24px
- Scales: 24, 48, 72, 96, etc
- Ascender Max: 14px
- Descender Max: 12px

## Kerning
none
