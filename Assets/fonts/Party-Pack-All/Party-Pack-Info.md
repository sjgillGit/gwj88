# Party Pack
Party Pack is a set of 10+ 22pt pixel art fonts, all designed to be variations of the original (normal) font.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 22px
- Scales: 22, 44, 66, 88, etc
- Ascender Max: 9px
- Descender Max: 6px
