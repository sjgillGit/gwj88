# Cowboy Western
Cowboy Western is a 16pt pixel art font designed to have a rough and tough western aesthetic.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 16px
- Scales: 16, 32, 48, 64, etc
- Ascender Max: 5px
- Descender Max: 4px
